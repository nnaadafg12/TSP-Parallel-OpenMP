# Parallelizing Travelling Salesman Problem - OpenMP

## Project Overview

This project focuses on solving the Traveling Salesman Problem (TSP) using parallel approaches. The main objective is to evaluate how parallelization with OpenMP can improve the performance of a brute-force TSP solution by reducing execution time and better utilizing multi-core processors.

The project was developed as part of a parallel computing course, with emphasis on performance comparison, scalability, and practical application of shared-memory parallelism.


## Objectives

- Develop a parallel version using OpenMP.
- Compare execution time, speedup, and scalability between sequential and parallel implementations.
- Analyze the impact of thread count on performance.
